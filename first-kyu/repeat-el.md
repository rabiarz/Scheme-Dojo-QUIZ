# repeat-el

# question
- Create a function called repeat-el that takes a number (n) and an atom as its arguments and return a list of that atom added (n) times in it.
- If the definition is insufficient, please define it yourself.

# prerequisites

- (repeat-el 'number 'atom)

# example

```
(print (repeat-el `3 `apple)) 
; (apple apple apple)

(print (repeat-el `0 `a))
; ()
```
